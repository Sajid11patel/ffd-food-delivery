# ffd-food-delivery
A fully functional food delivery frontend application built with HTML, CSS, and JavaScript, featuring login system, dynamic menu display, and cart management for enhanced usability.

 # 🍔 FFD - Food Delivery Web App

A clean and responsive food delivery web application built using HTML, CSS, and JavaScript, featuring user authentication, interactive menu browsing, and a dynamic cart system for a smooth user experience.

---

## 🚀 Features

- 🔐 User Signup/Login (LocalStorage)
- 🍕 Dynamic Food Menu (Categories)
- 🛒 Add to Cart System
- 💰 Total Price Calculation
- 📦 Order Confirmation Page
- 📞 Contact Page
- 🎯 Smooth Navigation
